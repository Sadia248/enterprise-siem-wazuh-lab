# Architecture

This folder contains the architecture diagrams and network design used in the Enterprise SIEM and SOC Monitoring Lab.

Components include:

- Wazuh SIEM Server
- Windows Endpoint
- Ubuntu Server
- Kali Linux Attack Machine
- Azure Virtual Networks (VNets)
- Network Security Groups (NSGs)
- VNet Peering
- MITRE ATT&CK Mapping

The architecture follows a segmented enterprise network design to support centralized monitoring, threat detection, and attack simulation.
